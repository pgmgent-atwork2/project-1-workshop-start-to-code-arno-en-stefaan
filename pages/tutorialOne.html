Stap 4: âœï¸ Schrijf condities

Nu onze buttons een functie hebben, kunnen we elke functie uitschrijven. Deze 2 functies zullen bijna exact het zelfde zijn behalve de conditie. Een conditie is een stukje code waarvan we gaan checken of het juist of fout is. Dit gaan we doen met een "if-else loop". 

              
//functies aanmaken voor als er op een knop gedrukt word
function clickHoger() {
  const nieuw = nieuwGetal();
  if (nieuw > huidigGetal) {
    huidigeScore++;
    score.textContent = huidigeScore;
  } else {
    aantalLevens--;
    levens.textContent = aantalLevens;
  }
  huidigGetal = nieuw;
  gokCijfer.textContent = huidigGetal;
  checkGameOver();
}

function clickLager() {
  const nieuw = nieuwGetal();
  if (nieuw < huidigGetal) {
    huidigeScore++;
    score.textContent = huidigeScore;
  } else {
    aantalLevens--;
    levens.textContent = aantalLevens;
  }
  huidigGetal = nieuw;
  gokCijfer.textContent = huidigGetal;
  checkGameOver();
}
