Visual Studio Code

Visual Studio Code is een programma waarmee je code kunt schrijven. 🪄 Het is speciaal gemaakt om bestanden te bewerken en mappen te organiseren. Met deze editor kun je zelf websites, apps en nog veel meer maken.

Je kunt ook handige extensies toevoegen om VS Code helemaal aan te passen zoals jij wilt. Zo maak je jouw werkomgeving precies zoals jij prettig vindt.

Klaar om te beginnen? Download Visual Studio Code ⏬ en start vandaag nog met bouwen!

Download Visual Studio Code
  • ☑️ Installeer het programma door het installatiebestand te openen en de stappen te volgen.
  • ☑️ Start VSCode zodra de installatie klaar is.
  • ☑️ 🔌Installeer in VSCode de extensie "Live Server"
  • ☑️ Als je wilt, kun je nog andere extensies toevoegen (maar dat is niet verplicht om te starten).
  • ☑️ Open je VScode langs de linkerkant, en je browser langs de rechterkant.
  • ☑️ ➡️ Druk op "Go Live" rechts onderaan.
  • ☑️ Klik op de Terug-knop in deze webapp.
  • ☑️ 🎯Druk op de Start-knop om je project te openen en direct te beginnen met bouwen in VS Code!
